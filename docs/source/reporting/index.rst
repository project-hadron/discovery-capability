Capability Reports
==================

Across all capability, Project Hadron provides a common set of reporting tools to provide insight
into where, what, how and why. It provides the facility to to acquire knowledge from SMEs and
report on that knowledge. It provides data lineage reports and reports on the veracity of the data
pass through the capability provided data profiling reports as a summary, data dictionary or
detailed schema.
